# Video Cuter

一个纯前端、本地处理的视频切片工具。页面侧重快速框选片段、跨视频整理片段、导出音频或视频，并支持把多个片段按顺序合并成一个最终文件。

## 主要能力

- 多视频导入，在同一工作区里切片和切换素材
- 基于波形图拖拽创建片段
- 批量导出单段视频、单段音频
- 将多个片段一次性合并导出为单个 MP4
- 语音转文字、字幕导出、双语字幕导出
- 本地保存工作区状态，刷新后可恢复同名素材的片段信息

## 近期更新

- 合并导出改为单次 `filter_complex` 编码，减少中间片段转码与读写
- 右侧片段面板改为更紧凑的摘要卡片，同时保留全部操作按钮
- 片段拖动和缩放冲突时改为边界吸附，不再直接回弹
- 为导出和波形交互补充 Node 测试

## 项目结构

- `index.html`: 主界面与布局
- `js/`: 前端模块
- `tests/`: Node 原生测试
- `docs/superpowers/specs/`: 设计文档
- `docs/superpowers/plans/`: 实施计划

## 本地运行

这是一个静态页面项目，直接用任意本地静态服务器打开仓库目录即可。应用依赖浏览器环境、FFmpeg WASM 和页面中的 CDN 脚本。

语音转写和 LLM 相关功能需要在页面设置里配置对应服务地址与凭证。
